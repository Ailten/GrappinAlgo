
# Grappin Algo

Réalisé par Ailten, inspiré de CitizenCode.

## Présentation

Grappin Algo est un petit jeu, pour apprendre le code (Python) en s'amusant.

Vous trouverez dans le fichier ``main.py`` (a la racine), une démonstration d'un première exercice.
Les ligne 3 a 20 crée l'exercice (tableau de départ et tableau attendu).
Ta mission cera de re créer le tableau attendu en déplacent les blocks avec le grappin ``^``.
Tu peu déplacer le grappin de gauche a droite avec la function ``droite()`` et ``gauche()``.
Tu peu également demander au grappin de prendre un block avec ``prendre()`` et le poser ``poser()``.

Attention a ne pas déplacer le grappin hors du tableau, ou a ne pas lui demander de prendre un block si il n'y a rien en dessous de lui.

Bonne chance.

## Exercices.

Pour les formateur, n'ésitez pas a créer de nouveau exercices de votre choix en modifiant les tableau de départ et d'arriver.
Assurez vous seulement que le resultat sois réalisable.

Quelque example d'exercices sont placé dans le docier Exercices, vous pouvez copier-coller la function ``makeExercice`` d'un fichier exercice, pour la placer dans le fichier ``main.py`` (n'oubliez pas de retirer l'encien makeExercice déja présent).
